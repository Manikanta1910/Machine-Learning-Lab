 **House Price Prediction using Linear Regression**

## Aim
To build a Machine Learning model that predicts house prices using Linear Regression based on given dataset features.

## Description
In this experiment, a regression model is developed to predict house prices using historical housing data.  
The dataset contains features such as house size, number of rooms, and other attributes that influence the price.

Linear Regression is used to learn the relationship between the input features and the target variable (house price).

## Algorithm Used
Linear Regression

## Tools and Libraries
- Python
- Pandas
- NumPy
- Matplotlib
- Scikit-learn
- Google Colab

## Steps Involved
1. Import required libraries.
2. Load the housing dataset.
3. Perform data preprocessing.
4. Split the dataset into training and testing sets.
5. Train the Linear Regression model.
6. Predict house prices using the trained model.
7. Evaluate model performance.

## Output
The model predicts house prices based on the given input features and visualizes the relationship between features and target values.

## Conclusion
Linear Regression successfully models the relationship between input features and house prices, allowing predictions for new data.
